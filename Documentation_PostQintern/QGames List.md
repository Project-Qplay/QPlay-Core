# Educational Quantum Games
Compiled by our QResearch intern team, this list focuses on the benefits of multiple quantum games/simulations for the purpose of developing our own project.

Games chosen and analyzed by QPlay members; list summarized by Perplexity.


## Games Overview

### AgentQ
[AgentQ](https://vicpina.itch.io/agent-q)

| Topic | Analysis |
| --- | --- |
| Type | Pico‑8 puzzle game focused on quantum error correction  |
| Quantum Concepts Covered | <ul><li>Quantum error correction basics via “find the odd one out and correct it” mechanics </li><li>Use of CX gates for copying qubit states and building redundancy </li><li>High‑level idea that quantum computers are error‑prone and require monitoring and correction </li></ul> |
| Potential Adaptations for QPlay | <ul><li>Error‑correction themed rooms where players must detect and fix corrupted bits/qubits to unlock doors </li><li>Visual CX/controlled‑operation puzzles that show how information propagates in codes </li><li>Layered “basic story mode” plus an “advanced E.C. lab” that dives deeper into real codes </li></ul> |
| Effective Learning Benefits | <ul><li>Simple UI and sound design make a technical topic approachable and memorable </li><li>Mechanics focus attention on the core idea of detecting and fixing inconsistencies </li></ul> |
| Areas of Improvement | <ul><li>Does not cover realistic error models, perturbations, or modern code families in depth </li><li>Conceptual depth is limited; feels more like an intro than a serious learning tool </li></ul> |
